# How to Run:

```bash
git clone https://github.com/OmarThinks/github-task/
cd github-task/react-native-app
npm install
```

### Running on Android:

```bash
npm run android
```

### Running on web:

```bash
npm run web
```

## Texhnologies used:

- React, React Router
- Redux, Redux Toolkit (RTK), RTK Query
- React Native, React Native Paper
